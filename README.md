# SCSS Bitirme Projesi

Bu proje, SCSS kullanılarak geliştirilmiş bir web sitesidir. SCSS ile yazılmış stiller, daha düzenli ve yönetilebilir bir CSS yapısı sağlar. Responsive tasarımı sayesinde farklı cihazlarda sorunsuz çalışır.

---
## Kurulum

Projeyi kullanmak için öncelikle GitHub reposunu bilgisayarınıza klonlayın: `git clone https://github.com/belinaykaraasln-beep/SCSS-Bitirme-Projesi.git`. Daha sonra proje dizinine girin: `cd SCSS-Bitirme-Projesi`. SCSS dosyalarını CSS’e derlemek için terminalde `sass --watch scss:css` komutunu kullanabilirsiniz. Siteyi tarayıcıda açmak için `index.html` dosyasını çift tıklayabilir veya terminalden Mac için `open index.html`, Windows için `start index.html` komutunu kullanabilirsiniz.

---

## Kullanım

- SCSS dosyaları ile stil düzenlemeleri yapılabilir.  
- Ana sayfada içerikler ve örnek stiller görüntülenir.  
- Responsive tasarım sayesinde farklı ekran boyutlarında çalışır.  
- CSS ve SCSS dosyalarını değiştirerek projeyi özelleştirebilirsiniz.

---

## Teknolojiler

- HTML5  
- SCSS / CSS3
